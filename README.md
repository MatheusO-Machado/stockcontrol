# StockControl (JavaFX + SQLite)

Sistema simples de controle de estoque feito em **Java 21** com **JavaFX 21** e (em breve) **SQLite**.

## Funcionalidades (em desenvolvimento)
- [x] Listagem de produtos (UI JavaFX)
- [ ] Cadastro/Edição/Exclusão de produtos
- [ ] Persistência com SQLite
- [ ] Busca e filtros
- [ ] Movimentações de entrada/saída

## Requisitos
- Java 21+
- Maven 3.9+
- JavaFX 21 (via Maven)

## Como executar
```bash
mvn clean javafx:run
```

## Tecnologias
- Java 21
- JavaFX 21
- Maven
- SQLite
